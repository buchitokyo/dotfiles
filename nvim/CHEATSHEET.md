# Neovim チートシート

## モード

| キー | モード |
|------|--------|
| `i` | 挿入モード（カーソル位置） |
| `a` | 挿入モード（カーソルの次） |
| `I` | 挿入モード（行頭） |
| `A` | 挿入モード（行末） |
| `o` | 下に新行追加して挿入モード |
| `O` | 上に新行追加して挿入モード |
| `v` | ビジュアルモード |
| `V` | 行ビジュアルモード |
| `Ctrl+v` | 矩形ビジュアルモード |
| `Esc` / `jj` | ノーマルモードに戻る |

## 移動

| キー | 機能 |
|------|------|
| `h` / `j` / `k` / `l` | 左 / 下 / 上 / 右 |
| `w` | 次の単語先頭へ |
| `b` | 前の単語先頭へ |
| `e` | 単語末尾へ |
| `H` | 行頭へ（カスタム） |
| `L` | 行末へ（カスタム） |
| `gg` | ファイル先頭へ |
| `G` | ファイル末尾へ |
| `Ctrl+d` | 半ページ下スクロール |
| `Ctrl+u` | 半ページ上スクロール |
| `Ctrl+o` | 前のジャンプ位置に戻る |
| `Ctrl+i` | 次のジャンプ位置に進む |
| `s` + 文字 + ラベル | Flash ジャンプ（画面内の任意の位置へ） |

## 編集

| キー | 機能 |
|------|------|
| `x` | 1文字削除 |
| `dd` | 行削除 |
| `dw` | 単語削除 |
| `D` | カーソルから行末まで削除 |
| `cc` | 行を削除して挿入モード |
| `cw` | 単語を削除して挿入モード |
| `r` | 1文字置換 |
| `.` | 直前の操作を繰り返す |
| `u` | 元に戻す（Undo） |
| `Ctrl+r` | やり直し（Redo） |
| `>>` / `<<` | インデント追加 / 削除 |

## コピー（ヤンク）& ペースト

| キー | 機能 |
|------|------|
| `yy` | 行コピー |
| `yw` | 単語をコピー |
| `y$` | カーソルから行末までコピー |
| `yi"` | `"..."` の中身をコピー |
| `yi'` | `'...'` の中身をコピー |
| `yi(` | `(...)` の中身をコピー |
| `yi{` | `{...}` の中身をコピー |
| `yi[` | `[...]` の中身をコピー |
| `yiw` | カーソル上の単語をコピー |
| `p` | カーソルの後に貼り付け |
| `P` | カーソルの前に貼り付け |

## 検索 & 置換

| キー | 機能 |
|------|------|
| `/検索語` | 前方検索 |
| `?検索語` | 後方検索 |
| `n` / `N` | 次 / 前の検索結果 |
| `*` | カーソル下の単語を前方検索 |
| `#` | カーソル下の単語を後方検索 |
| `:%s/old/new/g` | ファイル全体で置換 |
| `:%s/old/new/gc` | 確認しながら置換 |
| `Esc Esc` | 検索ハイライト解除 |

## ファイル操作（カスタム）

| キー | 機能 |
|------|------|
| `Space w` | 保存 |
| `Space q` | 終了 |
| `Space x` | 保存して終了 |
| `;` | コマンドモード（`:` の代わり） |
| `:wa` | 全ファイル保存 |
| `:qa` | 全ウィンドウ終了 |

## ウィンドウ操作

| キー | 機能 |
|------|------|
| `Ctrl+h/j/k/l` | ウィンドウ間移動 |
| `:sp` | 水平分割 |
| `:vsp` | 垂直分割 |
| `:q` | ウィンドウを閉じる |

## バッファ操作

| キー | 機能 |
|------|------|
| `Space n` | 次のバッファ |
| `Space p` | 前のバッファ |
| `Space d` | バッファを閉じる |

## Snacks Picker（ファジー検索）

| キー | 機能 |
|------|------|
| `Space Space` | スマートピッカー |
| `Space ff` | ファイル検索 |
| `Space fg` | 文字列検索（grep） |
| `Space fw` | カーソル下の単語を grep |
| `Space fb` | バッファ検索 |
| `Space fh` | ヘルプ検索 |
| `Space fr` | LSP 参照 |
| `Space fs` | ドキュメントシンボル |
| `Space sr` | 最近開いたファイル |
| `Space sc` | コマンド検索 |
| `Space sk` | キーマップ検索 |
| `Space sd` | 診断情報検索 |
| `Space sR` | 直前のピッカーを再開 |

## oil.nvim（ファイルエクスプローラー）

| キー | 機能 |
|------|------|
| `-` / `Space e` | ファイルエクスプローラーを開く |
| `Enter` | ファイル / ディレクトリを選択 |
| `-` | 親ディレクトリに戻る |
| `q` | 閉じる |
| ファイル名を編集 → `:w` | リネーム |
| `dd` → `:w` | ファイル削除 |

## LSP

| キー | 機能 |
|------|------|
| `gd` | 定義へジャンプ |
| `gr` | 参照一覧 |
| `K` | ホバードキュメント |
| `Space rn` | リネーム |
| `Space ca` | コードアクション |
| `[d` / `]d` | 前 / 次の診断メッセージ |
| `Ctrl+o` | ジャンプ前の位置に戻る |

## AI

| キー | 機能 |
|------|------|
| `Space ai` | Sidekick |
| `Space aa` | Avante |

## Git

| キー | 機能 |
|------|------|
| `Space gg` | LazyGit |

## その他

| キー | 機能 |
|------|------|
| `Space ?` | キーバインド一覧（which-key） |
| `Space a` | 全選択 |
| `Space ev` | 設定ファイルを開く |
| `Space sv` | 設定ファイルを再読み込み |
| `Space ss` | 末尾の空白を削除 |
| `gcc` | 行コメント切り替え |
| `gc`（ビジュアルモード） | 選択範囲をコメント切り替え |
